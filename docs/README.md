# TalkTwirl Web App 🌐

Welcome to **TalkTwirl** - a modern, feature-rich social media application now available on the web!

## 🚀 Live Demo

**Access the app**: [https://luohino.github.io/Talktwirl/](https://luohino.github.io/Talktwirl/)

## ✨ Features

- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 💬 **Real-time Messaging** - Chat with friends instantly
- 📸 **Post Sharing** - Share photos and videos
- 👥 **Social Connections** - Follow users and build your network
- 🔔 **Live Notifications** - Stay updated with real-time alerts
- 🎨 **Modern UI** - Beautiful dark theme interface
- 🔄 **Progressive Web App** - Install on your device like a native app

## 📱 Mobile Apps

- **Android APK**: [Download TalkTwirl v2.0.0](https://github.com/Luohino/Talktwirl/releases/tag/v2.0.0)
- **Universal Compatibility** - Works on all Android devices

## 🛠️ Technology Stack

- **Frontend**: Flutter Web
- **Backend**: Supabase
- **Real-time**: WebSocket connections
- **Storage**: Cloud storage for media
- **Authentication**: Email & Google Sign-in
